
------How to Run the PHP Scripts--------

 Description

This README provides instructions on how to set up a local environment to run PHP scripts on your computer. It includes guidance on installing a web server, configuring the environment, and running PHP scripts. 

------------Prerequisites-------------

- A web server with PHP support (e.g., Apache, Nginx, or a local development server like XAMPP, WAMP, or MAMP).
- A text editor or an integrated development environment (IDE) for writing PHP code.
- A web browser to view the output of the PHP scripts.

----------Installation and Setup--------------

1. Download and Install a Local Development Server:
   - XAMPP: [Download XAMPP](https://www.apachefriends.org/index.html)


2. Install the Development Server:
   - Follow the installation instructions provided on the respective websites.
   - Ensure that Apache and PHP are installed and configured correctly.

3. Start the Development Server:
   - For XAMPP:
     - Open the XAMPP Control Panel.
     - Click "Start" next to Apache.
     - Click "Start" next to Mysql.
  

-----------Extracting and Running PHP Scripts------------

1. Extract the ZIP file :
   - Copy and Paste the Extracted Folder to the web server's document root directory:
   Example:  "\C:\xampp\htdocs\WEB II (Sandesh Gadal)"


4. Run the Script in a Browser:
   - Open your web browser.
   - Navigate to "http://localhost/WEB%20II%20(SANDESH%20GADAL)/"
